# Ontwerp

Vorige weken zijn wij druk bezig geweest met het ontwerpen van een robot.
De keuze is gemaakt om de robot op rupsbanden te laten rijden,
zodat we het ruige maanterrein kunnen overwinnen.

![Rendering van de robot](/RoboticaPromotie/assets/robotRendered.png)

Ook moeten we met de robot stenen en bekers op kunnen pakken,
dus willen we een grijparm op de robot plaatsen.
Hierbij moeten we rekening houden met de lasers die achterop zitten.
Deze lasers scannen de omgeving, zodat de robot zelf zijn route kan bepalen.

# Code

Omdat de robot veel verschillende dingen moet kunnen doen,
moet er code worden geschreven om de robot te laten rijden,
alle sensoren uit te lezen, de arm te bewegen en meer.

Wij zijn druk bezig met de beginselen van de software,
zodat deze straks kan worden toegepast in de robot.
Omdat we nog onderzoek aan het plegen zijn naar hoe de softwarearchitectuur in elkaar moet zitten,
zijn we voor nu de accessoires aan het programmeren, zoals de camera's.
